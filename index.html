五步驟建立您的 PressPlay 簽名檔

1. 填寫您的基本資料

{{ name }}
{{ jobTitle }}
phone: (02) 2558-8355 #{{ extension }}
cellphone: {{ cellphone }}
email:  {{ email }}
Line ID: {{ lineId }}
103 台北市大同區西寧北路62-5號2樓

2. 點擊複製簽名檔

3. 至 GMail 設定頁面

點擊右側齒輪 進入 GMail 設定頁面

或

點擊 GMail 主畫面右上角 齒輪圖示，並點擊 查看所有設定

4. 設定您的簽名檔

a. 在 ＧMail 設定頁面中找到簽名設定，並點擊 建立新標籤
b. 新增一個簽名
c. 點擊空白區塊，ctrl + v (cmd + v) 貼上您的簽名檔
d. 將簽名檔設為預設值
e. 儲存設定

5. 完成！

恭喜你完成您的簽名檔，趕快找一個人來測試寄信看看吧，如果不知道要寄給誰，歡迎寄給 gg@pressplay.cc
